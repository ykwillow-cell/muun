# 무운 페르소나 기반 기능 구현 계획

## Persona 1: 습관적 확인러 "김루틴" (24세, 직장인)
**핵심 니즈**: 빠른 속도, 익명성, 간단함

### 구현 기능:
1. **빠른 로드 시간 최적화**
   - 로컬 스토리지에 저장된 정보로 1초 내 결과 표시
   - 스켈레톤 로딩 UI 제거, 즉시 결과 노출
   - 애니메이션 최소화

2. **간단한 결과 요약**
   - 올해운세: 한 문장 요약 + 월별 아이콘만 표시
   - 복잡한 설명 숨김 (토글로 펼치기)

3. **개인정보 보호 강조**
   - "저장되지 않습니다" 배너 표시
   - 로컬 스토리지만 사용 (서버 저장 안 함)

## Persona 2: 불안한 취준생 "이간절" (29세, 취준생)
**핵심 니즈**: 구체적인 정보, 시간 투자 가능, 보상형 광고

### 구현 기능:
1. **목적성 있는 테마형 운세**
   - "합격운", "이직운", "면접운" 등 카테고리 추가
   - 특정 목표에 맞춘 해석 제공

2. **상세한 텍스트 콘텐츠**
   - 월별 상세 해석 (현재: 한 줄 → 개선: 2-3문장)
   - "이 달에 주의할 점", "기회의 시기" 등 구체적 조언

3. **보상형 광고 준비**
   - "더 자세한 운세 보기" 버튼 → 15초 광고 시청
   - 광고 후 추가 콘텐츠 언락

---

## Persona 3: 관계중심 공유러 "박인싸" (21세, 대학생)
**핵심 니즈**: 예쁜 결과 카드, 간편한 공유, 친구 궁합

### 구현 기능:
1. **결과 카드 이미지 생성**
   - 궁합 결과를 1장의 아름다운 카드로 변환
   - 다운로드 + 카카오톡 공유 버튼

2. **친구 궁합 빠른 입력**
   - "친구 생일 입력" 버튼 → 팝업에서 간단 입력
   - 로그인 없이 즉시 결과

3. **공유 최적화**
   - 카드 이미지 미리보기 표시
   - SNS 공유 시 썸네일 자동 생성
