# 구글 애드센스 심사 통과 개선 계획

## 1. 필수 페이지 및 정책 (심사 거절 주요 원인)

### 1.1 개인정보처리방침 (Privacy Policy)
- **상태**: ✅ 기존 페이지 있음 (`/privacy`)
- **확인 사항**:
  - 쿠키 사용 명시
  - 제3자 광고 (Google AdSense) 정책 명시
  - 개인정보 수집 목적 명확화
  - 사용자 권리 설명

### 1.2 이용약관 (Terms of Service)
- **상태**: ✅ 기존 페이지 있음 (`/terms`)
- **확인 사항**:
  - 서비스 이용 규칙 명시
  - 금지 행위 명시
  - 면책 조항 포함

### 1.3 문의/연락처 페이지 (Contact Us)
- **상태**: ❌ 없음 (추가 필요)
- **필요 정보**:
  - 이메일 주소
  - 연락 양식
  - 응답 시간 안내

### 1.4 About 페이지 (회사/서비스 소개)
- **상태**: ✅ 기존 페이지 있음 (`/about`)
- **확인 사항**:
  - 서비스 목적 명확화
  - 신뢰성 강화 (팀 정보, 경력 등)

## 2. 콘텐츠 품질 요구사항

### 2.1 원본 콘텐츠 (Original Content)
**현재 상태**: 각 페이지의 콘텐츠 길이 부족

| 페이지 | 현재 단어 수 | 권장 단어 수 | 상태 |
|--------|------------|-----------|------|
| Home | ~300 | 500-800 | ⚠️ 부족 |
| About | ~400 | 500-1000 | ⚠️ 부족 |
| Manselyeok | ~200 | 400-600 | ⚠️ 부족 |
| Lifelong Saju | ~100 | 400-600 | ❌ 부족 |
| Yearly Fortune | ~100 | 400-600 | ❌ 부족 |
| Compatibility | ~100 | 400-600 | ❌ 부족 |

**개선 방안**:
- 각 서비스 페이지에 상세한 설명 추가
- 사주/운세 관련 교육 콘텐츠 추가
- FAQ 섹션 추가
- 블로그 또는 자료실 섹션 추가

### 2.2 중복 콘텐츠 (Duplicate Content) 회피
- 각 페이지가 고유한 가치 제공
- 메타 설명 고유화
- 페이지별 키워드 차별화

### 2.3 저작권 및 라이선스
- 모든 이미지 저작권 확인
- 마스코트 이미지 라이선스 확인
- 필요시 라이선스 표시

## 3. 기술적 요구사항

### 3.1 SEO 최적화
- **메타 태그**: 각 페이지에 고유한 title, description 필요
- **구조화된 데이터**: Schema.org 마크업 추가
- **모바일 최적화**: ✅ 이미 완료
- **페이지 속도**: PageSpeed Insights 90점 이상 목표

### 3.2 보안 (Security)
- **HTTPS**: ✅ Vercel 자동 제공
- **SSL 인증서**: ✅ 자동 갱신

### 3.3 사이트 구조
- **Sitemap.xml**: 추가 필요
- **Robots.txt**: 추가 필요
- **Navigation**: ✅ 명확함

---

## 4. 애드센스 정책 준수

### 4.1 금지 콘텐츠
- ❌ 폭력, 혐오 콘텐츠
- ❌ 성인 콘텐츠
- ❌ 도박, 불법 활동
- ❌ 저작권 침해
- ✅ 무운은 모두 준수

### 4.2 광고 배치 정책
- 클릭 유도 금지 (클릭하라는 텍스트 금지)
- 광고 주변에 명확한 공간 필요
- 광고와 콘텐츠 구분 필요

### 4.3 트래픽 요구사항
- **신규 사이트**: 최소 6개월 운영 권장
- **월간 방문자**: 최소 1,000-10,000 권장
- **현재 상태**: 신규 사이트 → 콘텐츠 강화 필수

## 5. 개선 우선순위

### Phase 1: 필수 (심사 거절 방지)
	1. ✅ Privacy Policy 업데이트 (Google AdSense 정책 추가 및 쿠키 고지 구체화)
	2. ✅ Terms of Service 확인
	3. ✅ Contact Us 페이지 추가 및 라우팅 연결 확인
	4. ✅ Sitemap.xml 생성 및 등록
	5. ✅ Robots.txt 생성 및 Googlebot 허용 설정

### Phase 2: 중요 (심사 통과 확률 향상)
	1. ✅ 각 페이지 콘텐츠 확대 (Home, About, Tojeong 보강 완료)
	2. ✅ 메타 태그 최적화 (기존 설정 확인)
	3. ❌ 블로그/자료실 섹션 추가 (향후 과제)
	4. ✅ FAQ 섹션 추가 (Contact 및 Home에 포함)

### Phase 3: 권장 (심사 통과 후)
1. Schema.org 구조화된 데이터 추가
2. PageSpeed Insights 최적화
3. Google Search Console 등록
4. Google Analytics 설정
