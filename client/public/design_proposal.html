개선 포인트 상세 설명

1

배너: 흰 카드 → 컬러 그라디언트 이미지형 카드

현재 배너는 흰 배경에 텍스트만 있어 와이어프레임처럼 보입니다. 개선안은 각 배너마다 고유한 컬러 그라디언트(황토·세이지·인디고·차콜)를 적용하고, 배경에 한자 레이어를 반투명하게 깔아 시각적 깊이감을 줍니다. CTA 버튼도 유리 질감(glassmorphism)으로 처리합니다.

핵심 변경 gradient 한자 레이어 glassmorphism CTA
2

Hero: 배경 앰비언트 + 황토 강조색

흰 배경에 미세한 방사형 그라디언트(황토/베이지 계열)를 추가하여 단순한 흰 박스에서 벗어납니다. 제목의 "내 사주" 부분에 황토 강조색(#6b5c3e)을 적용하고, 상단에 "명리학 기반 운세" eyebrow 텍스트를 추가하여 브랜드 정체성을 강화합니다.

ambient gradient eyebrow label accent color
3

서비스 카드: 배경 한자 워터마크

각 서비스 카드 우하단에 해당 서비스를 상징하는 한자(合·命·星·占)를 매우 연한 투명도로 배치합니다. 텍스트 정렬도 가운데 → 왼쪽 정렬로 변경하여 더 정보 밀도 있는 느낌을 줍니다.

한자 워터마크 좌측 정렬
4

작명소 Spotlight: 다크 카드로 전환

현재 흰 카드로 된 Spotlight를 다크(#1a1a18) 배경으로 전환하여 섹션 간 시각적 리듬을 만듭니다. 내부에 미세한 황토 앰비언트를 추가하고, 한자 "字"를 반투명 배경 요소로 사용합니다.

핵심 변경 dark card visual rhythm
5

더보기 그리드: 아이콘+텍스트 수평 레이아웃

현재 더보기 카드는 아이콘이 위, 텍스트가 아래인 수직 구조로 빈 공간이 많습니다. 개선안은 아이콘 박스를 왼쪽에, 서비스명+서브텍스트를 오른쪽에 배치하는 수평 레이아웃으로 변경하여 정보 밀도를 높이고 서브텍스트도 추가합니다.

수평 레이아웃 서브텍스트 추가 정보 밀도 향상
6

바로가기: Hero 하단에 통합

바로가기 4버튼을 Hero 영역 바로 아래(흰 배경 연속)에 배치하여 Hero와 자연스럽게 연결되도록 합니다. 아이콘 박스에 #f5f4ef 배경을 적용하여 시각적으로 구분합니다.

Hero 통합 아이콘 박스
구현 난이도: 모든 변경사항은 기존 CSS 변수 시스템을 유지하면서 적용 가능합니다. 배너 그라디언트는 MainBanner.tsx의 인라인 스타일 또는 CSS 클래스로, 서비스 카드 한자 워터마크는 data-glyph 속성 + CSS ::before로 구현합니다.
