01 — 배경 시스템
토스 배경 #F2F4F6
토스는 순백 배경 대신 연회색 #F2F4F6을 페이지 배경으로 사용합니다. 카드는 흰색 #FFFFFF로 배경 위에 자연스럽게 떠오르는 레이어 구조를 만듭니다. 이것이 "카드가 공중에 떠있는" 느낌의 핵심입니다.
bg: #F2F4F6 card: #FFFFFF radius: 16px
02 — 타이포그래피
3단계 텍스트 위계
토스는 텍스트를 3단계로 구분합니다. Primary(#191F28) 핵심 정보, Secondary(#4E5968) 보조 정보, Tertiary(#8B95A1) 힌트/레이블. 이 3단계만으로 모든 UI의 위계를 표현합니다.
#191F28 primary #4E5968 secondary #8B95A1 tertiary
03 — 포인트 컬러
무운 그린 #03C75A
토스 블루(#3182F6) 대신 무운의 아이덴티티 컬러인 네이버 그린 #03C75A를 포인트로 사용합니다. CTA 버튼, 로고 dot, eyebrow 레이블, 배지, 홈 탭 인디케이터에만 절제하여 사용합니다.
#03C75A CTA badge nav dot eyebrow
04 — 아이콘
SVG 라인 아이콘
이모지(💑🔮🃏)를 완전히 제거하고 stroke-width 1.6px의 SVG 라인 아이콘으로 교체합니다. 아이콘은 #F2F4F6 배경의 40×40px 박스 안에 배치하여 일관된 크기와 시각적 리듬을 만듭니다.
stroke: 1.6px icon-box: 40px radius: 12px
05 — 배너
컬러 그라디언트 카드
흰 배경 배너 카드를 완전히 제거하고 서비스별 컬러 그라디언트 배경으로 교체합니다. 반투명 한자 워터마크(運·家·合·名)가 배경에 배치되어 시각적 깊이를 만듭니다. 가로 스크롤로 탐색합니다.
green 신년운세 blue 가족사주 amber MBTI purple 작명소
06 — Bottom Nav
라이트 탭바 + 그린 dot
토스의 흰 배경 탭바 스타일을 그대로 적용합니다. 활성 탭은 아이콘 우상단에 작은 그린 dot(4px)으로 표시합니다. 이모지 아이콘 대신 SVG 라인 아이콘을 사용합니다.
bg: #FFFFFF border-top: #E8EBED active dot: #03C75A
