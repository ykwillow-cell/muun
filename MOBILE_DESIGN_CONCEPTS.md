# 무운(MUUN) 모바일 메인 페이지 최적화 - 디자인 시안 2가지

## 현재 문제점
- Hero Section이 85vh로 너무 길어서 스크롤 없이는 메뉴와 추천 서비스가 보이지 않음
- 모바일 유입이 많은 사이트 특성상 **진입 직후 즉시 서비스 접근성이 떨어짐**
- 일반적인 모바일 사이트 UX와 맞지 않음

---

## 시안 1: 콤팩트 Hero + 즉시 메뉴 노출

### 레이아웃 구조
```
┌─────────────────────────────────┐
│  Header (GNB)                   │ ← 고정
├─────────────────────────────────┤
│                                 │
│  Hero Section (40vh)            │ ← 축소
│  - 배경 이미지 (오로라)          │
│  - 타이틀 + 간단한 설명          │
│  - CTA 버튼 (2026 신년운세)     │
│                                 │
├─────────────────────────────────┤
│ 카테고리 칩 (가로 스크롤)        │ ← 즉시 노출
│ [전체] [사주] [타로] [궁합]...  │
├─────────────────────────────────┤
│ 추천 서비스 (카로셀)             │ ← 즉시 노출
│ [2026신년운세] [평생사주]...   │
├─────────────────────────────────┤
│ 서비스 그리드 (2열)              │
│ [사주] [타로] [궁합]...         │
└─────────────────────────────────┘
```

### 특징
- **Hero 높이**: 40vh (모바일에서 약 360px)
- **배경**: 오로라 이미지 + 그라데이션 오버레이
- **타이틀**: "당신의 운명을 비추는 가장 맑은 거울, 무운" (2줄)
- **설명**: 간단한 한 줄 설명만 표시
- **CTA**: 버튼 1개 (2026 신년운세)
- **오늘의 키워드 카드**: 제거 또는 매우 축소
- **스크롤 없이 보이는 영역**: Hero + 카테고리 칩 + 추천 서비스 일부

### 장점
- 진입 직후 **메뉴와 서비스가 즉시 보임**
- 일반적인 모바일 사이트 UX와 유사
- 빠른 서비스 접근성
- 깔끔하고 직관적

### 단점
- 오늘의 키워드/행운 컬러 정보가 축소됨
- 개인화 느낌이 약할 수 있음

---

## 시안 2: 미니 Hero + 대시보드 카드 우선

### 레이아웃 구조
```
┌─────────────────────────────────┐
│  Header (GNB)                   │ ← 고정
├─────────────────────────────────┤
│                                 │
│  Hero Section (35vh)            │ ← 더 축소
│  - 배경 이미지 (오로라)          │
│  - 타이틀만 (한 줄)              │
│  - CTA 버튼                     │
│                                 │
├─────────────────────────────────┤
│ 오늘의 키워드 카드 (작은 크기)   │ ← 즉시 노출
│ ┌─────────────────────────────┐ │
│ │ 오늘의 키워드: 변화           │
│ │ 행운 컬러: 로즈 골드          │
│ │ 행운 숫자: 33                 │
│ └─────────────────────────────┘ │
├─────────────────────────────────┤
│ 카테고리 칩 (가로 스크롤)        │
│ [전체] [사주] [타로] [궁합]...  │
├─────────────────────────────────┤
│ 추천 서비스 (카로셀)             │
│ [2026신년운세] [평생사주]...   │
├─────────────────────────────────┤
│ 서비스 그리드 (2열)              │
│ [사주] [타로] [궁합]...         │
└─────────────────────────────────┘
```

### 특징
- **Hero 높이**: 35vh (모바일에서 약 315px)
- **배경**: 오로라 이미지 + 진한 그라데이션 오버레이
- **타이틀**: "당신의 운명을 비추는 가장 맑은 거울, 무운" (한 줄로 축약 가능)
- **설명**: 제거 또는 매우 축소
- **CTA**: 버튼 1개 (2026 신년운세)
- **오늘의 키워드 카드**: 더 작고 간결하게 (높이 80px 정도)
- **스크롤 없이 보이는 영역**: Hero + 오늘의 키워드 카드 + 카테고리 칩 일부

### 특징
- **개인화 정보 우선**: 오늘의 키워드/행운 컬러를 즉시 노출
- **더 빠른 서비스 접근**: Hero가 더 작아서 스크롤 최소화
- **신비로운 느낌 유지**: 오로라 배경과 개인화 정보의 조화
- **재방문율 증대**: 매일 달라지는 정보로 인한 재방문 유도

### 단점
- Hero가 매우 작아서 브랜드 임팩트가 약할 수 있음
- 타이틀이 너무 축약되면 가독성 저하

---

## 비교 표

| 항목 | 시안 1 (콤팩트 Hero) | 시안 2 (미니 Hero) |
|------|------------------|-----------------|
| Hero 높이 | 40vh | 35vh |
| 개인화 정보 | 축소/제거 | 우선 노출 |
| 브랜드 임팩트 | 중상 | 중 |
| 서비스 접근성 | 높음 | 매우 높음 |
| 재방문율 | 중 | 높음 |
| 모바일 UX | 일반적 | 최적화됨 |
| 추천 대상 | 신규 유입 중심 | 재방문 유도 중심 |

---

## 권장사항

**사용자의 모바일 유입이 많고 재방문을 중시한다면 → 시안 2 (미니 Hero + 대시보드 카드 우선)**

이유:
1. 매일 달라지는 오늘의 키워드/행운 컬러로 **재방문 유도**
2. 스크롤 최소화로 **즉시 서비스 접근 가능**
3. 개인화 정보로 **사용자 몰입도 증대**
4. 모바일 최적화된 UX

**신규 유입을 중시한다면 → 시안 1 (콤팩트 Hero + 즉시 메뉴 노출)**

이유:
1. 깔끔하고 직관적인 레이아웃
2. 일반적인 모바일 사이트 UX
3. 빠른 서비스 탐색 가능
